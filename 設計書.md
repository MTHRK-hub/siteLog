## 5. 共通仕様

### SCR-00-00 フレームページ

`index.html` 起動時に最初に開く画面。frameを使用してヘッダ（SCR-00-01）と各コンテンツ画面を同時に表示する。

**フレーム領域:**

| 領域 | 内容 | ファイル |
|------|------|----------|
| ① ヘッダ領域 | 共通ヘッダを表示 | SCR-00-01.html |
| ② コンテンツ領域 | 各画面のHTMLを表示 | 各画面HTML（SCR-01-01〜SCR-99-01） |

**ファイル構成:**

| ファイル | 役割 |
|----------|------|
| SCR-00-00.html | フレーム構成を定義するHTMLファイル。ヘッダ領域（SCR-00-01）とコンテンツ領域（各画面）をframeで構成 |

### SCR-00-01 共通ヘッダ

ヘッダは各画面固有に実装するのではなく、共通ファイル（`SCR-00-01.html` / `SCR-00-01.css` / `SCR-00-01.js`）として定義し、全画面から読み込む。

**ファイル構成:**

| ファイル | 役割 |
|----------|------|
| SCR-00-01.html | フレームの①（ヘッダ領域）に表示する独立したHTMLファイル |
| SCR-00-01.css | ヘッダのスタイル定義 |
| SCR-00-01.js | ヘッダの動的制御（表示要素の切り替え・ログアウト処理など） |

**ヘッダ要素:**

| # | 要素 | 位置 | 表示条件 | 説明 |
|---|------|------|----------|------|
| ① | ログインユーザー名 | 左上 | SCR-01-01以外 | ユーザー情報から取得したログインユーザーのユーザー名 |
| ② | メニューに戻るボタン | 左下 | SCR-03-01・SCR-04-01・SCR-05-01・SCR-06-01・SCR-07-01・SCR-08-01・SCR-A1-01のみ | 押下でSCR-02-01へ |
| ② | 一覧に戻るボタン | 左下 | SCR-03-02・SCR-04-02・SCR-05-02・SCR-06-02・SCR-08-02のみ | 押下で各一覧画面（SCR-03-01・SCR-04-01・SCR-05-01・SCR-06-01・SCR-08-01）へ（SCR-06-02では「計画画面に戻る」と表示） |
| ③ | 画面ID | 中央上 | 全画面 | 各画面のID（文字サイズは画面名の半分程度） |
| ④ | 画面名 | 中央 | 全画面 | 各画面のタイトル |
| ⑤ | ログアウトボタン | 右上 | SCR-01-01以外 | 押下で「ログアウトしますか？」のダイアログを表示 → 「はい」でSCR-01-01へ遷移 |
| ⑥ | 新規作成/新規登録ボタン | 右下 | SCR-03-01・SCR-04-01・SCR-05-01・SCR-06-01・SCR-07-01・SCR-08-01・SCR-A2-01のみ | 押下で各登録画面（SCR-03-03・SCR-04-03・SCR-05-03・SCR-06-03）へ（SCR-07-01ではキャッシュフロー登録画面（後日実装）へ、SCR-08-01では**SCR-08-03 イベント登録画面**へ、SCR-A2-01では新規ユーザー登録フォームへ） |

## 6. 画面詳細仕様

### SCR-01-01 ログイン画面

**ヘッダ:** 画面ID・画面名のみ（ログインユーザー名・ログアウトボタンは非表示）

**コンポーネント:**

| 要素 | 種別 | 説明 |
|------|------|------|
| ユーザーID | 入力欄 | ユーザーID入力 |
| パスワード | 入力欄（マスク） | パスワード入力 |
| ログインボタン | ボタン | 認証処理を実行 |
| 閉じるボタン | ボタン | フレームページ（SCR-00-00）ごとウィンドウを閉じる |

**ロジック:**

1. 入力されたユーザーIDとパスワードを**ユーザー情報**と照合（スプレッドシートから取得したパスワードは復号化して照合する）
2. 一致するレコードが存在する場合 → **SCR-02-01 メニュー画面**へ遷移
3. 一致しない場合 → エラーを表示（遷移しない）

> **スプレッドシートから取得したパスワードを復号化してログイン照合処理を行う**

**ヘッダ:** 画面ID・画面名・ログインユーザー名（左上）・ログアウトボタン（右上）・**メニューに戻るボタン（左下）**・**新規登録ボタン（右下）**

**データ取得:** 画面遷移時に**友達情報**（2行目以降）を取得して一覧に表示。ユーザーIDがログインユーザーと合致するレコードのみ表示する。**名前の昇順**で表示する。

| # | 要素 | 種別 | 説明 |

| ① | プルダウン | プルダウン | 選択肢：`全件表示`（初期値・全件表示）、`20代`、`予定あり`。`20代` を選択した場合は年齢が20〜29のデータのみ表示。`予定あり` を選択した場合は今後の予定が `あり:〜` のデータのみ表示。 |
| ② | 友達一覧テーブル | テーブル | 友達一覧を表示 |

友達一覧テーブル

| カラム | 説明 |
|--------|------|
| 名前 | リンク形式で表示。押下で **SCR-03-02 友達詳細画面**へ遷移 |
| 年齢 | 下記ロジックで算出した値を表示 |
| 性別 | |
| 職業 | |

**年齢の算出ロジック:**

| 条件 | 表示内容 |
|------|----------|
| 対象データの**生年月日**が入力されている | 生年月日から現在の年齢を算出して表示 |
| 生年月日が未入力かつ**年齢差**が入力されている | ログインユーザーの生年月日と年齢差から年齢を算出して表示 |
| 生年月日・年齢差ともに未入力 | 表示しない（空欄） |

**0件時の表示:** 表示対象データが0件の場合、テーブル内に「データがありません」と表示する。

### SCR-A1-01 パスワード変更画面

**ヘッダ:** 画面ID・画面名・ログインユーザー名（左上）・ログアウトボタン（右上）・**メニューに戻るボタン（左下）**

**コンポーネント:**

| 要素 | 種別 | 説明 |
|------|------|------|
| パスワード入力欄 | 入力欄（マスク） | 現在のパスワードを入力 |
| 送信ボタン | ボタン | 現在のパスワードを照合し、正しければ新しいパスワード入力欄を表示 |
| 新しいパスワード入力欄 | 入力欄（マスク） | パスワード入力欄の照合が成功した場合に表示 |
| 新しいパスワード（確認用）入力欄 | 入力欄（マスク） | パスワード入力欄の照合が成功した場合に表示 |
| 変更ボタン | ボタン | 「パスワードを変更しますか？」の確認ダイアログを表示 |

**ロジック:**

1. 「パスワード入力欄」に現在のパスワードを入力し「送信ボタン」押下
2. ユーザー情報シートのパスワードと照合
   - 正しい場合 → 新しいパスワード・新しいパスワード（確認用）の入力欄を表示
   - 正しくない場合 → エラーを表示（遷移しない）
3. 「変更ボタン」押下時 → 「パスワードを変更しますか？」の確認ダイアログを表示
4. 「はい」押下 → ユーザー情報シートの対象ユーザーのパスワードを暗号化して更新 → **SCR-99-01 完了画面**へ遷移

### SCR-A2-01 ユーザー一覧画面

**ヘッダ:** 画面ID・画面名・ログインユーザー名（左上）・ログアウトボタン（右上）・**新規登録ボタン（右下）**

**データ取得:** スプレッドシートの「ユーザー情報」シートからデータを取得して一覧に表示

**コンポーネント:**

ユーザー一覧テーブル

| カラム | 説明 |
|--------|------|
| ユーザーID | ユーザーのログインID |
| ユーザー名 | ユーザー名 |
| 管理権限 | 管理者フラグが `1` の場合は「あり」、`0` の場合は「なし」と表示 |
| 初期化 | 初期化ボタン |
| 削除 | 削除ボタン |

**初期化ボタン:**

| 操作 | 説明 |
|------|------|
| 初期化ボタン押下 | 「（ユーザー名）を初期化しますか？」の確認ダイアログを表示 |
| 「はい」押下 | 対象ユーザーのパスワードを `resetPass` に更新 → **SCR-99-01 完了画面**へ遷移 → **SCR-A2-01 ユーザー一覧画面**へ遷移 |

**削除ボタン:**

| 操作 | 説明 |
|------|------|
| 削除ボタン押下 | 「（ユーザーID）\_（ユーザー名）さんを削除しますか？」の確認ダイアログを表示 |
| 「はい」押下 | スプレッドシートの対象ユーザーのユーザー情報を削除し、他のシート（友達情報・現場記録情報・メモ情報・企画情報・キャッシュフロー情報・イベント情報）に登録されている対象ユーザーのユーザーIDのデータも全て削除 → **SCR-99-01 完了画面**へ遷移 → **SCR-A2-01 ユーザー一覧画面**へ遷移 |

### SCR-A2-02 ユーザー登録画面

**ヘッダ:** 画面ID・画面名・ログインユーザー名（左上）・ログアウトボタン（右上）・**一覧に戻るボタン（左下）**

**アクセス制限:** 管理者（管理者フラグ = `1`）のみ表示可。非管理者はメニュー画面へリダイレクト。

**入力項目:**

| 項目 | 種別 | 備考 |
|------|------|------|
| ユーザーID | テキスト | 必須 |
| ユーザー名 | テキスト | 必須 |
| 生年月日 | 日付 | 任意 |
| 管理者権限 | ラジオボタン | 「なし」（0）・「あり」（1）、デフォルト：なし |

**コンポーネント:**

| ボタン | 説明 |
|--------|------|
| 登録ボタン | 「登録しますか？」の確認ダイアログを表示。「はい」押下でスプレッドシートのユーザー情報に行を追加（パスワードは `newPass` を暗号化して登録） → **SCR-99-01 完了画面**へ遷移 |
| キャンセルボタン | 入力を破棄して**SCR-A2-01 ユーザー一覧画面**へ戻る |
